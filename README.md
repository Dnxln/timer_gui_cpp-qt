Execute program in Linux:
1. mkdir < folder >
2. cd < folder >
3. git clone https://github.com/Dnxln/timer_gui_cpp-qt
4. cd timer_gui_cpp-qt/
5. sudo apt update
6. sudo apt install build-essential cmake qtbase5-dev qt5-qmake qtbase5-dev-tools
7. mkdir build
8. cd build
9. cmake ..
10. make
11. ./AppLauncher
